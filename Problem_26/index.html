PICT B.E. Course Structure

First Year (FE)

  • Mathematics I – Calculus, differential equations.
  • Physics – Mechanics, optics, thermodynamics.
  • Engineering Drawing – Basics of projections and CAD.
  • Programming in C – Basic programming constructs.

Second Year (SE)

  • Data Structures – Arrays, stacks, trees, graphs.
  • Digital Electronics – Logic gates, flip flops, multiplexers.
  • Object-Oriented Programming – Concepts of classes, inheritance.
  • Database Management Systems – SQL, normalization.

Third Year (TE)

  • Operating Systems – Processes, memory management, scheduling.
  • Computer Networks – OSI model, TCP/IP, routing.
  • Web Technologies – HTML, CSS, JS, backend basics.
  • Software Engineering – SDLC, Agile methodologies.

Final Year (BE)

  • Machine Learning – Supervised/unsupervised learning, models.
  • Cloud Computing – Virtualization, AWS basics.
  • Project Work – Capstone team-based software project.
  • Electives – Blockchain, IoT, Cybersecurity, etc.
